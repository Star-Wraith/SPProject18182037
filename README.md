# SPProject18182037
쿠키 대전쟁

## 🎮 게임 컨셉
냥코 대전쟁의 게임방식과 쿠키런 리소스가 합쳐진 디펜스 게임입니다.


## 개발 범위
- 총 3Stage
- Player 쿠키 10종
- 적 쿠키 20종(Stage당 6종, 7종, 7종 출현 예정)
- StartMap, 쿠키 엔트리 설정 Map(Ready Map), InGameMap등 button 약 10개
- Timer
